Global Scope

var, let, const declared outside of a function
var gVar = "My Global Var"; let gLet = "My Global Let"; const gConst = "My Global Const"; console.log(`${window.gVar} ${window.gLet} ${window.gConst}`);

Function/Local Scope

let, const declared inside of a function
//These are invalid console.log(fLet); console.log(fConst); //This is valid console.log(fVar); function myFunction() { var fVar = "My Local Var"; let fLet = "My Local Let"; const fConst = "My Local Const"; console.log(fVar); console.log(fLet); console.log(fConst); } //These are invalid console.log(fLet); console.log(fConst); //This is valid console.log(fVar);
