document.querySelector(selector)

CSS Selector syntax
Get first element with id `output`:
document.querySelector("#output")
Get first div element:
document.querySelector("div")
Get first element where the attribute `data-id` equals `data-output`:
document.querySelector("[data-id=data-output]")
Get first element with class `myClass`:
document.querySelector(".myClass")

document.querySelectorAll(name)

Get all elements with attribute `data-id`:
document.querySelectorAll("[data-id]")
Get all div elements:
document.querySelectorAll("div")
Get all elements with attribute class equal to `myClass`:
document.querySelectorAll(".myClass")
